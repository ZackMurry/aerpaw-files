Simple jammer for use with USRPs. The following options can be used at runtime.
You can select a radio by using --args "serial=[device serial]" or --args "addr=[device ip addr]".
  -h, --help            show the help message and exits
  --args ARGUMENTS,     Set arguments [default='']
  -d DURATION, --duration DURATION
                        Set duration [default=0]
  -f FREQ, --freq FREQ  Set freq [default='2.635G']
  -g GAIN, --gain GAIN  Set gain [default='0.0']
  -r RATE, --rate RATE  Set rate [default='2.0M']
