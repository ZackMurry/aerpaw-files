
Usage:
main('folder')
where 'folder' is the folder that contains the following two files
GPS_DATA.csv
experiment.txt

Example: Run the code below to plot exp1 and exp2 results on top of each other and save the results as jpg format
close all
main('exp1')
main('exp2') 
legends.m % to add legends to the plots generated by main.m and saves them as jpg format.

KML file generation
main.m creates input.csv inside the 'folder'. 
Usage
python3 kml_generator.py --input exp1/input.csv --output exp1/exp1.kml
